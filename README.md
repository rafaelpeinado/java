### O que é Java?
- Linguagem de Programação
- API
- Ambiente de Execução


#### Linguagem de Programação
- Simples
- Orientada a Objetos
- Distribuída
- Multithreaded
- Dinâmica
- Arquitetura neutra
- Portável
- Ótima Performance
